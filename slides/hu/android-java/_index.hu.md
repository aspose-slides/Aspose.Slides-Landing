<p>
 Az Aspose.Slides for Android a Java-n keresztül egy PowerPoint API Android-alkalmazásokhoz. A fejlesztők a prezentációfeldolgozási feladatok széles skáláját hajthatják végre, beleértve a prezentációk olvasását, írását, módosítását, egyesítését, felosztását, konvertálását és manipulálását anélkül, hogy a Microsoft PowerPointtól függenének.
</p>

<p>
 Az Aspose.Slides for Android a Java segítségével szinte az összes natív PowerPoint fájlformátumot, valamint az OpenOffice prezentációs formátumot (ODP) támogatja. Hozzon létre PowerPoint megjelenítő alkalmazást Androidra, valamint konvertert, szerkesztőt vagy bármilyen mást. A bemutatóelemek, például a SmartArt-alakzatok, az MSO-diagramok, az ActiveX-összetevők és az OLE-keretobjektumok a PowerPoint API-n keresztül érhetők el kezelés és kezelés céljából.
</p>
