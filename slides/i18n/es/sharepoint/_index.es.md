<p>
 Aspose.Slides for SharePoint para administrar y convertir archivos de PowerPoint en SharePoint. Aspose.Slides, como una solución de PowerPoint de SharePoint, puede importar fácilmente formatos de presentación comunes de Microsoft, incluidos PPT, POT, PPS, PPTX, POTX y PPSX, y puede exportar estos formatos a archivos PDF, TIFF y XPS. Utilice toda la funcionalidad de SharePoint PPT sin necesidad de Microsoft PowerPoint Automation.
</p>

<p>
 Aspose.Slides para SharePoint funciona con Windows SharePoint Services WSS 3.0 y Microsoft Office SharePoint Server 2007, 2010 y 2013.
</p>
