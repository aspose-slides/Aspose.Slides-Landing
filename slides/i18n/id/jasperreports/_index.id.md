<p>
 Aspose.Slides for JasperReports adalah API PowerPoint untuk ekspor laporan JasperReports dan JasperServer yang fleksibel dan mudah digunakan ke PowerPoint. JasperServer tidak memiliki kemampuan bawaan untuk mengekspor laporan sebagai presentasi PowerPoint. Namun, dengan API PowerPoint ini, laporan JasperServer dapat diekspor ke format presentasi PPT, PPS, PPTX, dan PPSX tanpa menggunakan Microsoft PowerPoint.
</p>
