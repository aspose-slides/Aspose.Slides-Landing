<p>
 Aspose.Slides for JasperReports هو PowerPoint API لتصدير مرن وسهل الاستخدام لتقارير JasperReports و JasperServer إلى PowerPoint. لا يمتلك JasperServer القدرة المضمنة على تصدير التقارير كعروض PowerPoint التقديمية. ومع ذلك ، باستخدام PowerPoint API ، يمكن تصدير تقارير JasperServer إلى تنسيقات العروض التقديمية PPT و PPS و PPTX و PPSX دون استخدام Microsoft PowerPoint.
</p>
