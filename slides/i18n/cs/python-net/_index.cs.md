<p><strong>Aspose.Slides pro Python přes .NET</strong> je výkonná knihovna tříd pro práci s prezentacemi nebo jejich zpracování. Pomocí tohoto produktu mohou aplikace a vývojáři číst, upravovat nebo manipulovat a převádět prezentace PowerPoint (PPT, PPTX) a prezentace v jiných formátech (ODP) bez aplikací nebo závislostí třetích stran. </p>

<p><strong>Aspose.Slides pro Python prostřednictvím. NET</strong> poskytuje tyto oblíbené funkce:</p>
<ul>
    <li>Načítání, otevírání a prohlížení prezentací.</li>
    <li>Úprava prezentací.</li>
    <li>Převod prezentací do PDF, Word, JPG, HTML, GIF, SVG a mnoha dalších formátů.</li>
    <li>Rendering a tisk prezentací.</li>
    <li>Šifrování a dešifrování prezentací; heslem chráněných prezentací a odstraňování hesel.</li>
    <li>Manipulace s entitami prezentace, jako jsou hlavní snímky, tvary, grafy, rámečky obrázků, zvukové rámečky, rámečky videa, OLE, makra VBA, animace atd.</li>
    <li>A mnoho dalších funkcí.</li>
</ul>

<p>Python je velmi populární jazyk, který nachází mnoho využití v běžných aplikacích, vývoji webu, výzkumných a akademických úkolech, analýze dat atd. import aspose.slides as slides

import aspose.slides as slides

with slides.Presentation("Presentation1.pptx") as pres1:
    with slides.Presentation("Presentation2.pptx") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
        pres1.save("combined.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>

import aspose.slides as slides

with slides.Presentation() as pres:
    pres.slides.remove_at(0)
    pres.slides.add_from_pdf("welcome-to-powerpoint.pdf")
    pres.save("OutputPresentation.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">

        <pre>
            <code class="python">
import aspose.slides as slides

# Instantiate a Presentation object that represents a PPT file
presentation = slides.Presentation("PowerPoint.ppt")

# Save the presentation as PDF
presentation.save("PPT-to-PDF.pdf", slides.export.SaveFormat.PDF)
            </code>
        </pre>

        <h2 class="h2title">Převeďte PowerPoint do JPG: Python PPTX, PPT nebo ODP na JPEG</h2>

        <pre>
            <code class="python">
import aspose.slides as slides
import aspose.pydrawing as drawing

pres = slides.Presentation("pres.pptx")

for sld in pres.slides:
    bmp = sld.get_thumbnail(1, 1)
    bmp.save("Slide_{num}.jpg".format(num=str(sld.slide_number)), drawing.imaging.ImageFormat.jpeg)
            </code>
        </pre>
