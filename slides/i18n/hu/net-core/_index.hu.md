<p>
 Az Aspose.Slides for .NET Core prezentációfeldolgozási megoldások létrehozására használható bármilyen operációs rendszeren: Windows, Linux vagy macOS. Prezentáció és elemeinek létrehozása és módosítása: szöveg, táblázatok, diagramok, média, animációk, SmartArts és ActiveX elemek.
</p>

<p>
 Bővítse alkalmazásait prezentációnézegetővel, készítővel, konvertálóval, szerkesztővel, sablonkészítővel, kezelővel, egyesítővel, összehasonlítóval, elosztóval, nyomtatóval, védővel és másokkal. Lehetővé teszi a C# alkalmazások számára, hogy módosítsák a prezentáció tartalmát: szövegek, táblázatok, diagramok, média, SmartArts, ActiveX.
</p>

<p>
 Az Aspose.Slides .NET Core PowerPoint API egyedülálló lehetőséget kínál a kódduplikáció elkerülésére és a .NET Core platform minden erejének kihasználására. Az API a PowerPointtól vagy az Open Office-tól függetlenül működik, és nincs szükség rájuk.
</p>

<p>
</p>

    <h2 class="h2title">
     Prezentáció konvertálása .NET Core-ban
    </h2>
    <p>
     Az Aspose.Slides for .NET Core API bármilyen PowerPoint prezentációs formátumot konvertálhat dokumentum-, kép-, web- vagy fájlformátummá. Néhány népszerű konverziós formátum: PDF, HTML, JPEG, XPS. A prezentációs formátumok egymás között is konvertálhatók (például PPT-ből PPTX-be, PPTX-ből ODP-be), vagy sablonformátumba (például PPTX-ből POTM-be), diavetítés-formátumba (pl. PPTX-ből PPSX-be) és fordítva. A konvertált fájl minősége a legmagasabb az Aspose.Slides által a PowerPoint .NET Core API-hoz fejlesztett renderelő motornak köszönhetően. Biztos lehet benne, hogy bármilyen prezentációt bármilyen operációs rendszerre konvertál: Windows, Linux vagy macOS.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Prezentációk renderelése - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>

A prezentációtervezés sablon, téma, diaminta, diaelrendezés segítségével hozható létre. A média, a vízjel, az ActiveX az Aspose.Slides által a PowerPoint .NET Core API-n keresztül is elérhető.
    </p>

Ezek közé tartozik a prezentációk és sablonok összehasonlítása, a prezentációk felosztása az egyéni logika alapján, a különböző formátumú prezentációk összevonása, a sablonok egyesítése/alkalmazása prezentációk készletébe, prezentációk nyomtatása stb.
