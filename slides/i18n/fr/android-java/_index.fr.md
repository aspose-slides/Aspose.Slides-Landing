<p>
 Aspose.Slides pour Android via Java est une API PowerPoint pour les applications Android. Les développeurs peuvent effectuer un large éventail de tâches de traitement de présentation, notamment la lecture, l'écriture, la modification, la fusion, le fractionnement, la conversion et la manipulation de présentations sans aucune dépendance à Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides pour Android via Java prend en charge presque tous les formats de fichiers PowerPoint natifs ainsi que le format de présentation OpenOffice (ODP). Créez votre application de visionneuse PowerPoint pour Android, ainsi qu'un convertisseur, un éditeur ou tout autre. Les éléments de présentation tels que les formes SmartArt, les graphiques MSO, les composants ActiveX et les objets de cadre OLE sont accessibles via l'API PowerPoint pour la gestion et la manipulation.
</p>

TiffOptions opts = new TiffOptions();                

INotesCommentsLayoutingOptions options = opts.getNotesCommentsLayouting();

//Saving to TIFF notes

pres.save(dataDir + "TestNotes.tiff", SaveFormat.Tiff,opts);
