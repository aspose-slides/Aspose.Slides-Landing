<p><strong>Aspose.Slides voor Python via .NET</strong> is een krachtige klassenbibliotheek voor het werken met of het verwerken van presentaties. Met dit product kunnen applicaties en ontwikkelaars PowerPoint-presentaties (PPT, PPTX) en presentaties in andere formaten (ODP) lezen, bewerken of manipuleren en converteren zonder applicaties of afhankelijkheden van derden. </p>

<p><strong>Aspose.Slides voor Python via. NET</strong> biedt deze populaire functies:</p>
<ul>
    <li>Presentaties laden, openen en bekijken.</li>
    <li>Presentaties bewerken.</li>
    <li>Presentaties converteren naar PDF, Word, JPG, HTML, GIF, SVG en vele andere formaten.</li>
    <li>Presentaties maken en afdrukken.</li>
    <li>Versleutelen en ontsleutelen van presentaties; presentaties met wachtwoord beveiligen en wachtwoorden verwijderen.</li>
    <li>Manipuleren van presentatie-entiteiten, zoals basisdia's, vormen, grafieken, fotolijsten, audioframes, videoframes, OLE, VBA-macro's, animaties, enz.</li>
    <li>En nog veel meer functies.</li>
</ul>

<p>Python is een zeer populaire taal die veel wordt gebruikt in reguliere toepassingen, webontwikkeling, onderzoeks- en academische taken, gegevensanalyse, enz. Om deze reden is het <strong>Aspose.Slides</strong>-team trots om <strong> strong>Aspose.Slides voor Python via .NET</strong> aan de python-community. </p>

    <div class="col-lg-12">
        <h2 class="h2title">Maak een nieuwe PowerPoint-presentatie. Python PPTX, PPT of ODP</h2>
        <p>In het onderstaande voorbeeld hebben we een regel toegevoegd aan de eerste dia van de presentatie.</p>
        <pre>
            <code class="python">
import aspose.slides as slides

# Instantiate a Presentation object that represents a presentation file
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
    slide.shapes.add_auto_shape(slides.ShapeType.LINE, 50, 150, 300, 0)
    presentation.save("NewPresentation_out.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>

    <div class="col-lg-12">
        <h2 class="h2title">Presentaties samenvoegen: Python PPTX, PPT of ODP</h2>
        <p>Deze Python-code laat zien hoe je presentaties samenvoegt:</p>
        <pre>
            <code class="python">
import aspose.slides as slides

with slides.Presentation("Presentation1.pptx") as pres1:
    with slides.Presentation("Presentation2.pptx") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
        pres1.save("combined.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title"> Presentatie importeren vanuit PDF: Python PPTX, PPT of ODP</h2>
        <p>Deze Python-code demonstreert het conversieproces van PDF naar PowerPoint:</p>
        <pre>
            <code class="python">
import aspose.slides as slides

with slides.Presentation() as pres:
    pres.slides.remove_at(0)
    pres.slides.add_from_pdf("welcome-to-powerpoint.pdf")
    pres.save("OutputPresentation.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Converteer PowerPoint naar PDF met standaardopties: Python PPTX, PPT of ODP naar PDF</h2>
        <p>Deze python-code laat zien hoe u een PowerPoint PPT-, PPTX- en OpenOffice ODP-document converteert naar een PDF-document met behulp van de standaardopties. Het resulterende bestand is een PDF-document met de maximale kwaliteitsniveaus</p>
        <pre>
            <code class="python">
import aspose.slides as slides

# Instantiate a Presentation object that represents a PPT file
presentation = slides.Presentation("PowerPoint.ppt")

# Save the presentation as PDF
presentation.save("PPT-to-PDF.pdf", slides.export.SaveFormat.PDF)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Converteer PowerPoint naar JPG: Python PPTX, PPT of ODP naar JPEG</h2>
        <p>In het volgende voorbeeld ziet u hoe u een PowerPoint PPT-, PPTX- en OpenOffice ODP-document converteert naar een set JPEG-afbeeldingen.</p>
        <pre>
            <code class="python">
import aspose.slides as slides
import aspose.pydrawing as drawing

pres = slides.Presentation("pres.pptx")

for sld in pres.slides:
    bmp = sld.get_thumbnail(1, 1)
    bmp.save("Slide_{num}.jpg".format(num=str(sld.slide_number)), drawing.imaging.ImageFormat.jpeg)
            </code>
        </pre>
