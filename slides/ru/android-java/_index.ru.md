<p>
 Aspose.Slides для Android через Java — это API PowerPoint для приложений Android. Разработчики могут выполнять широкий спектр задач по обработке презентаций, включая чтение, запись, изменение, слияние, разделение, преобразование и управление презентациями без какой-либо зависимости от Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides для Android через Java поддерживает почти все родные форматы файлов PowerPoint, а также формат презентаций OpenOffice (ODP). Создайте свое приложение для просмотра PowerPoint для Android, а также конвертер, редактор или любое другое. К элементам презентации, таким как фигуры SmartArt, диаграммы MSO, компоненты ActiveX и объекты кадра OLE, можно получить доступ через API PowerPoint для управления и манипулирования.
</p>
