<p>
 Aspose.Slides for JasperReports — это API PowerPoint для гибкого и простого в использовании экспорта отчетов JasperReports и JasperServer в PowerPoint. JasperServer не имеет встроенной возможности экспорта отчетов в виде презентаций PowerPoint. Однако с помощью этого API PowerPoint отчеты JasperServer можно экспортировать в форматы презентаций PPT, PPS, PPTX и PPSX без использования Microsoft PowerPoint.
</p>
