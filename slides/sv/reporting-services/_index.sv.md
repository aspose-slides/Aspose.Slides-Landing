<p>
 Aspose.Slides for Reporting Services är den enda lösningen på marknaden som gör det möjligt att generera presentationer i PowerPoints ursprungliga format inifrån Microsoft SQL Server Reporting Services.
</p>

<p>
 Aspose.Slides for Reporting Services stöder nästan alla RDL-funktioner som tabeller, matriser, diagram, bilder, listor, underrapporter och sidhuvuden/sidfötter, vilket gör det möjligt att exportera rapporter i PowerPoint-presentationsformat på ett sådant sätt att de ser likadana ut som originalet rapportdesign. Utför Reporting Services SQL PPT utan Microsoft PowerPoint Automation.
</p>
