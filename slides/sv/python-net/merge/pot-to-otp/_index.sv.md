{{% blocks/products/pf/feature-page-section  h2="Slå samman POT-filer till OTP med Python" %}}
För att slå samman PowerPoint-presentationer måste du klona bilderna från en presentation till den andra.

{{% blocks/products/pf/agp/code-block title="Python-kod för att slå samman flera POT till en enda OTP-fil" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.pot") as pres1:
    with slides.Presentation("presentation2.pot") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.otp", slides.export.SaveFormat.OTP)
```
