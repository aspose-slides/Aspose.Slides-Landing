<p><strong>Aspose.Slides för Python via .NET</strong> är ett kraftfullt klassbibliotek för att arbeta med eller bearbeta presentationer. Med den här produkten kan applikationer och utvecklare läsa, redigera eller manipulera och konvertera PowerPoint-presentationer (PPT, PPTX) och presentationer i andra format (ODP) utan tredjepartsapplikationer eller beroenden. </p>

<p><strong>Aspose.Slides för Python via. NET</strong> tillhandahåller dessa populära funktioner:</p>
<ul>
    <li>Laddar, öppnar och visar presentationer.</li>
    <li>Redigera presentationer.</li>
    <li>Konvertera presentationer till PDF, Word, JPG, HTML, GIF, SVG och många andra format.</li>
    <li>Rendering och utskrift av presentationer.</li>
    <li>Kryptering och dekryptering av presentationer; lösenordsskyddande presentationer och ta bort lösenord.</li>
    <li>Manipulera presentationsenheter, såsom huvudbilder, former, diagram, bildramar, ljudramar, videoramar, OLE, VBA-makron, animationer, etc.</li>
    <li>Och många fler funktioner.</li>
</ul>

<p>Python är ett mycket populärt språk som ser mycket användning i vanliga applikationer, webbutveckling, forskning och akademiska uppgifter, dataanalys, etc. Av denna anledning är <strong>Aspose.Slides</strong>-teamet stolta över att kunna erbjuda < strong>Aspose.Slides för Python via .NET</strong> till python-gemenskapen. </p>

    <div class="col-lg-12">
        <h2 class="h2title">Skapa ny PowerPoint-presentation. Python PPTX, PPT eller ODP</h2>
        <p>I exemplet nedan har vi lagt till en rad i presentationens första bild.</p>
        <pre>
            <code class="python">
import aspose.slides as slides

# Instantiate a Presentation object that represents a presentation file
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
    slide.shapes.add_auto_shape(slides.ShapeType.LINE, 50, 150, 300, 0)
    presentation.save("NewPresentation_out.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>

        <h2 class="h2title">Slå samman presentationer: Python PPTX, PPT eller ODP</h2>
        <p>Denna Python-kod visar hur du slår samman presentationer:</p>
        <pre>
            <code class="python">
import aspose.slides as slides

with slides.Presentation("Presentation1.pptx") as pres1:
    with slides.Presentation("Presentation2.pptx") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
        pres1.save("combined.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title"> Importera presentation från PDF: Python PPTX, PPT eller ODP</h2>
        <p>Denna Python-kod demonstrerar PDF till PowerPoint-konverteringsprocessen:</p>
        <pre>
            <code class="python">
import aspose.slides as slides

with slides.Presentation() as pres:
    pres.slides.remove_at(0)
    pres.slides.add_from_pdf("welcome-to-powerpoint.pdf")
    pres.save("OutputPresentation.pptx", slides.export.SaveFormat.PPTX)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Konvertera PowerPoint till PDF med standardalternativ: Python PPTX, PPT eller ODP till PDF</h2>
        <p>Den här pythonkoden visar hur du konverterar ett PowerPoint PPT-, PPTX- och OpenOffice ODP-dokument till ett PDF-dokument med standardalternativen. Den resulterande filen är ett PDF-dokument med högsta kvalitetsnivåer</p>
        <pre>
            <code class="python">
import aspose.slides as slides

# Instantiate a Presentation object that represents a PPT file
presentation = slides.Presentation("PowerPoint.ppt")

# Save the presentation as PDF
presentation.save("PPT-to-PDF.pdf", slides.export.SaveFormat.PDF)
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Konvertera PowerPoint till JPG: Python PPTX, PPT eller ODP till JPEG</h2>
        <p>Följande exempel visar hur du konverterar ett PowerPoint PPT-, PPTX- och OpenOffice ODP-dokument till en uppsättning JPEG-bilder.</p>
        <pre>
            <code class="python">
import aspose.slides as slides
import aspose.pydrawing as drawing

pres = slides.Presentation("pres.pptx")

for sld in pres.slides:
    bmp = sld.get_thumbnail(1, 1)
    bmp.save("Slide_{num}.jpg".format(num=str(sld.slide_number)), drawing.imaging.ImageFormat.jpeg)
            </code>
        </pre>
