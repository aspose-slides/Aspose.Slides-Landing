<p>Aspose.Slides för .NET är ett Presentation Processing API för PowerPoint- och OpenOffice-format. Aspose.Slides gör det möjligt för applikationer att läsa, skriva, skydda, ändra och konvertera presentationer i .NET C#. Hantera presentationstext, former, diagram, tabeller och animationer, lägg till ljud och video till bilder, förhandsgranska bilder.</p>

<p>Aspose.Slides för .NET stöder alla populära presentationsformat. Använd C# för att slå samman, dela, klona och återanvända flera PowerPoint-presentationer, mallar och bilder på många olika sätt. Dessutom erbjuder API ett antal avancerade funktioner som att skriva ut på fysiska skrivare och rendera presentationer till format med fast layout, HTML och bilder.</p>

<p>Aspose.Slides för .NET är ett fristående API och kräver inte Microsoft PowerPoint eller någon annan extra programvara eller bibliotek.</p>
