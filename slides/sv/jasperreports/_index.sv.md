<p>
 Aspose.Slides för JasperReports är ett PowerPoint API för flexibel och lättanvänd export av JasperReports och JasperServer-rapporter till PowerPoint. JasperServer har inte den inbyggda förmågan att exportera rapporter som PowerPoint-presentationer. Men med detta PowerPoint API kan JasperServer-rapporterna exporteras till PPT, PPS, PPTX och PPSX presentationsformat utan att använda Microsoft PowerPoint.
</p>
