<p>
 Aspose.Slides untuk SharePoint untuk mengelola dan mengonversi file PowerPoint di SharePoint. Aspose.Slides, sebagai solusi SharePoint PowerPoint, dapat dengan mudah mengimpor format presentasi Microsoft yang umum termasuk PPT, POT, PPS, PPTX, POTX & PPSX, dan dapat mengekspor format ini ke file PDF, TIFF, dan XPS. Gunakan semua fungsionalitas SharePoint PPT tanpa diperlukan Microsoft PowerPoint Automation.
</p>

<p>
 Aspose.Slides untuk SharePoint berfungsi dengan Windows SharePoint Services WSS 3.0 dan Microsoft Office SharePoint Server 2007, 2010 & 2013.
</p>
