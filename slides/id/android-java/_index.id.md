<p>
 Aspose.Slides untuk Android melalui Java adalah API PowerPoint untuk aplikasi Android. Pengembang dapat melakukan berbagai tugas pemrosesan presentasi termasuk membaca, menulis, memodifikasi, menggabungkan, memisahkan, mengonversi, dan memanipulasi presentasi tanpa ketergantungan pada Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides untuk Android melalui Java mendukung hampir semua format file PowerPoint asli serta format presentasi OpenOffice (ODP). Buat aplikasi penampil PowerPoint Anda untuk Android, serta konverter, editor, atau lainnya. Elemen presentasi seperti bentuk SmartArt, bagan MSO, komponen ActiveX, dan objek bingkai OLE dapat diakses melalui API PowerPoint untuk manajemen dan manipulasi.
</p>
