<p>
 Aspose.Slides untuk .NET Core digunakan untuk membuat solusi pemrosesan presentasi pada sistem operasi apa pun: Windows, Linux, atau macOS. Buat dan ubah presentasi dan elemennya: teks, tabel, bagan, media, animasi, elemen SmartArts, dan ActiveX.
</p>

<p>
 Perluas aplikasi Anda dengan penampil presentasi, pembuat, pengonversi, editor, konstruktor templat, pengelola, penggabungan, pembanding, pembagi, pencetak, pelindung, dan lainnya. Izinkan aplikasi C# untuk memodifikasi konten presentasi apa pun: teks, tabel, bagan, media, SmartArts, ActiveX.
</p>

<p>
 .NET Core PowerPoint API oleh Aspose.Slides adalah kesempatan unik untuk menghindari duplikasi kode dan menggunakan semua kekuatan platform .NET Core. API bekerja secara independen dari PowerPoint atau Open Office dan tidak memerlukannya.
</p>

    <h2 class="h2title">
     Konversi Presentasi di .NET Core
    </h2>
    <p>
     Aspose.Slides untuk .NET Core API dapat mengonversi format presentasi PowerPoint apa pun menjadi format dokumen, gambar, web, atau file. Beberapa format konversi yang populer adalah: PDF, HTML, JPEG, XPS. Format presentasi juga dapat dikonversi antara satu sama lain (misalnya PPT ke PPTX, PPTX ke ODP), atau ke dalam format template (misalnya PPTX ke POTM), format tayangan slide (misalnya PPTX ke PPSX) dan sebaliknya. Kualitas file yang dikonversi adalah yang tertinggi berkat mesin rendering yang dikembangkan oleh Aspose.Slides untuk PowerPoint .NET Core API. Anda mungkin yakin untuk mengonversi presentasi apa pun di OS apa pun: Windows, Linux, atau macOS.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Membuat presentasi - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>

    </h2>
    <p>

Semua konten tekstual dapat diedit melalui elemen seperti teks, paragraf, porsi, tabel, catatan, komentar, simbol matematika. 2D, efek 3D dan animasi tersedia untuk bentuk, bagan, SmartArts, WordArts. Desain presentasi dapat dibuat menggunakan template, theme, slide master, slide layout. Media, watermark, ActiveX juga dapat diakses melalui PowerPoint .NET Core API oleh Aspose.Slides.
    </p>
