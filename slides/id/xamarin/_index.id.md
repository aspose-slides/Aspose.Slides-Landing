<p>
 Aspose.Slides for Xamarin adalah solusi lintas platform untuk membuat aplikasi seluler PowerPoint untuk perangkat dan OS apa pun. Izinkan aplikasi seluler Anda mengedit teks, media, dan bentuk slide, menyimpan presentasi dalam format yang diinginkan. Tinjau presentasi dari ponsel, tambahkan komentar dan catatan, dan kirimkan. Terapkan tema baru pada presentasi, pisahkan presentasi pada beberapa file.
</p>

<p>
 Aspose.Slides for Xamarin adalah Xamarin PowerPoint API yang berdiri sendiri, yang tidak memerlukan Microsoft PowerPoint atau perangkat lunak tambahan lainnya.
</p>

    </h2>
    <p>
     Aspose.Slides for Xamarin API memungkinkan pembuatan aplikasi seluler untuk konversi PowerPoint ke dalam dokumen, tekstual, gambar, web, atau format lainnya. Konversikan presentasi dengan kualitas tinggi bahkan dari ponsel. Konversikan presentasi ke format PDF, HTML, XPS, JPG. PowerPoint Xamarin API mudah digunakan, logika konversi diimplementasikan dalam kode 1-2 baris. Cukup muat presentasi dan simpan dalam format yang dibutuhkan.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Membuat presentasi - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
