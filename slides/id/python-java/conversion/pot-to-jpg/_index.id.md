Untuk mengonversi POT ke JPG, Anda perlu membuat Presentasi dari file POT dan menyimpannya sebagai JPG.

{{% blocks/products/pf/agp/code-block title="Tutorial Python untuk mengubah POT menjadi JPG" offSpacer="true" %}}

```python

import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat
from javax.imageio import ImageIO
from java.io import File

pres = Presentation("PowerPoint.pot");

for i in range(pres.getSlides().size()):
    buffImage = pres.getSlides().get_Item(i).getThumbnail(2, 2)
    ImageIO.write(buffImage, "JPEG", File("slide" + str(i) + ".jpg"))

jpype.shutdownJVM()
```
