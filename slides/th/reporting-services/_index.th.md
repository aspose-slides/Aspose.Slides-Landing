<p>
 Aspose.Slides for Reporting Services เป็นโซลูชันเดียวในตลาดที่ทำให้สามารถสร้างงานนำเสนอในรูปแบบดั้งเดิมของ PowerPoint จากภายใน Microsoft SQL Server Reporting Services

</p>
