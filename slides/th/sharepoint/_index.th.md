<p>
 Aspose.Slides สำหรับ SharePoint เพื่อจัดการและแปลงไฟล์ PowerPoint ใน SharePoint Aspose.Slides เป็นโซลูชัน SharePoint PowerPoint สามารถนำเข้ารูปแบบการนำเสนอของ Microsoft ทั่วไป เช่น PPT, POT, PPS, PPTX, POTX & PPSX และสามารถส่งออกรูปแบบเหล่านี้เป็นไฟล์ PDF, TIFF และ XPS ใช้ฟังก์ชัน SharePoint PPT ทั้งหมดโดยไม่จำเป็นต้องใช้ Microsoft PowerPoint Automation
</p>

<p>
 Aspose.Slides for SharePoint ทำงานร่วมกับ Windows SharePoint Services WSS 3.0 และ Microsoft Office SharePoint Server 2007, 2010 & 2013
</p>
