<p>
 Aspose.Slides para .NET Core se utiliza para crear soluciones de procesamiento de presentaciones en cualquier sistema operativo: Windows, Linux o macOS. Cree y cambie la presentación y sus elementos: texto, tablas, gráficos, medios, animaciones, SmartArts y elementos ActiveX.
</p>

<p>
 Amplíe sus aplicaciones con visor de presentaciones, creador, convertidor, editor, constructor de plantillas, administrador, fusión, comparador, divisor, impresora, protector y otros. Permita que las aplicaciones de C# modifiquen cualquier contenido de presentación: textos, tablas, gráficos, medios, SmartArts, ActiveX.
</p>

<p>
 La API de PowerPoint .NET Core de Aspose.Slides es una oportunidad única para evitar la duplicación de código y utilizar todo el poder de la plataforma .NET Core. La API funciona independientemente de PowerPoint u Open Office y no los requiere.
</p>

    <h2 class="h2title">
     Convertir presentación en .NET Core
    </h2>
    <p>
     Aspose.Slides para .NET Core API puede convertir cualquier formato de presentación de PowerPoint en un formato de documento, imagen, web o archivo. Algunos de los formatos de conversión populares son: PDF, HTML, JPEG, XPS. Los formatos de presentación también se pueden convertir entre sí (p. ej., PPT a PPTX, PPTX a ODP), o en un formato de plantilla (p. ej., PPTX a POTM), un formato de presentación de diapositivas (p. ej., PPTX a PPSX) y viceversa. La calidad del archivo convertido es la más alta gracias a un motor de renderizado desarrollado por Aspose.Slides para PowerPoint .NET Core API. Puede estar seguro de convertir cualquier presentación en cualquier sistema operativo: Windows, Linux o macOS.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Renderizar presentaciones - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>

     Modificar presentación en .NET Core
    </h2>
    <p>
     Con Aspose.Slides para .NET Core puedes modificar la lógica y el aspecto de la presentación. Todo el contenido textual se puede editar a través de elementos como texto, párrafo, porción, tabla, nota, comentario, símbolo matemático. Los efectos y animaciones 2D y 3D están disponibles para formas, gráficos, SmartArts y WordArts. El diseño de presentaciones se puede crear usando plantilla, tema, patrón de diapositivas, diseño de diapositivas. También se puede acceder a medios, marcas de agua, ActiveX a través de PowerPoint .NET Core API de Aspose.Slides.
    </p>

Entre ellos se encuentran la comparación de presentaciones y plantillas, la división de presentaciones según la lógica personalizada, la fusión de presentaciones de varios formatos, la fusión/aplicación de plantillas en un conjunto de presentaciones, la impresión de presentaciones, etc.
