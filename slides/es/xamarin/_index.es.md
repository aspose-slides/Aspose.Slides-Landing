<p>
 Aspose.Slides para Xamarin es una solución multiplataforma para crear aplicaciones móviles de PowerPoint para cualquier dispositivo y sistema operativo. Permita que su aplicación móvil edite texto, medios y formas de la diapositiva, guarde la presentación en el formato deseado. Revisa la presentación desde el móvil, agrega comentarios y notas, y envíala. Aplique un nuevo tema en la presentación, divida la presentación en varios archivos.
</p>

<p>
 Aspose.Slides para Xamarin es una API de PowerPoint de Xamarin independiente que no requiere Microsoft PowerPoint ni ningún otro software adicional.
</p>

    <h2 class="h2title">
     Convertir presentación a través de Xamarin PowerPoint API
    </h2>
    <p>
     Aspose.Slides for Xamarin API permite crear aplicaciones móviles para la conversión de PowerPoint en cualquier documento, texto, imagen, web u otros formatos. Convierte presentaciones con una gran calidad incluso desde un móvil. Convierte presentaciones a formato PDF, HTML, XPS, JPG. La API de PowerPoint Xamarin es fácil de usar, la lógica de conversión se implementa en un código de 1-2 líneas. Simplemente cargue una presentación y guárdela en el formato necesario.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Renderizar presentaciones - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>

Todas las operaciones disponibles en el contenido de la presentación se pueden aplicar a través de la API Xamarin de Aspose.Slides. Los comentarios de la presentación, las notas, las propiedades del documento, así como los diseños, los temas y las plantillas también están disponibles para modificar.

Por ejemplo, la funcionalidad de combinación se puede usar para aplicar una plantilla a una presentación, para combinar algunas plantillas, para combinar presentaciones de diferentes formatos en un solo formato, etc.
