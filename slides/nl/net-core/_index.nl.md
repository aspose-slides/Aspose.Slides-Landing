<p>
 Aspose.Slides voor .NET Core wordt gebruikt om oplossingen voor presentatieverwerking te maken op elk besturingssysteem: Windows, Linux of macOS. Maak en wijzig de presentatie en zijn elementen: tekst, tabellen, grafieken, media, animaties, SmartArts en ActiveX-elementen.
</p>

<p>
 Breid uw toepassingen uit met presentatieviewer, maker, converter, editor, sjabloonbouwer, manager, fusie, vergelijker, splitter, printer, beschermer en andere. Sta C#-toepassingen toe om elke presentatie-inhoud te wijzigen: teksten, tabellen, grafieken, media, SmartArts, ActiveX.
</p>

<p>
 De .NET Core PowerPoint API van Aspose.Slides is een unieke kans om codeduplicatie te voorkomen en alle kracht van het .NET Core-platform te gebruiken. De API werkt onafhankelijk van PowerPoint of Open Office en heeft deze niet nodig.
</p>

   <div class="col-lg-12">
    <h2 class="h2title">
     Presentatie converteren in .NET Core
    </h2>
    <p>
     Aspose.Slides voor .NET Core API kan elk PowerPoint-presentatieformaat converteren naar een document-, afbeeldings-, web- of bestandsformaat. Enkele van de populaire conversie-indelingen zijn: PDF, HTML, JPEG, XPS. Presentatieformaten kunnen ook onderling worden geconverteerd (bijv. PPT naar PPTX, PPTX naar ODP), of naar een sjabloonformaat (bijv. PPTX naar POTM), een diavoorstellingsformaat (bijv. PPTX naar PPSX) en vice versa. De kwaliteit van het geconverteerde bestand is het hoogst dankzij een rendering-engine die is ontwikkeld door Aspose.Slides voor PowerPoint .NET Core API. U kunt er zeker van zijn dat u elke presentatie op elk besturingssysteem kunt converteren: Windows, Linux of macOS.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Presentaties renderen - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Presentatie wijzigen in .NET Core
    </h2>
    <p>
     Met Aspose.Slides voor .NET Core kunt u de logica en het uiterlijk van de presentatie wijzigen. Alle tekstuele inhoud kan worden bewerkt via elementen als tekst, alinea, gedeelte, tabel, notitie, commentaar, wiskundig symbool. Er zijn 2D-, 3D-effecten en animaties beschikbaar voor vormen, grafieken, SmartArts, WordArts. Het ontwerp van presentaties kan worden gemaakt met behulp van een sjabloon, thema, diamodel, dia-indeling. Media, watermerk en ActiveX zijn ook toegankelijk via PowerPoint .NET Core API door Aspose.Slides.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">

Voorbeelden hiervan zijn het vergelijken van presentaties en sjablonen, het splitsen van presentaties op basis van de aangepaste logica, het samenvoegen van presentaties van verschillende formaten, het samenvoegen/toepassen van een sjabloon in een reeks presentaties, het afdrukken van presentaties, enz.
    </p>
