Met [**Aspose.Slides voor C++**](https://products.aspose.com/slides/nl/cpp/) kunt u met slechts een paar regels code een nieuwe regel tekst toevoegen aan een PDF-document.

{{% blocks/products/pf/agp/code-block title="C++-code voor het bewerken van PDF" offSpacer="true" %}}
```cpp

auto pres = System::MakeObject<Presentation>();

pres->get_Slides()->RemoveAt(0);
pres->get_Slides()->AddFromPdf(u"document.pdf");

auto slide = pres->get_Slides()->idx_get(0);
auto shape = slide->get_Shapes()->AddAutoShape(ShapeType::Rectangle, 10.0f, 10.0f, 100.0f, 50.0f);
shape->get_TextFrame()->set_Text(u"New text");

pres->Save(u"document.pdf", SaveFormat::Pdf);
```
