Met behulp van [**Aspose.Slides for .NET**](https://products.aspose.com/slides/nl/net/), kunt u een PNG-afbeelding converteren naar een PowerPoint-presentatie met slechts een paar regels code:

{{% blocks/products/pf/agp/code-block title="C#-code voor het converteren van PNG naar PPT" offSpacer="true" %}}
```cs
using (Presentation pres = new Presentation())
{
    IPPImage image = pres.Images.AddImage(File.ReadAllBytes("image.png"));
    pres.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 0, 0, 720, 540, image);
    pres.Save("Presentation.ppt", SaveFormat.Ppt);
}
```
