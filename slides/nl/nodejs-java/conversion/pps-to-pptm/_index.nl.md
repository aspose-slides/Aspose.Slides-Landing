Om het bestand PPS naar PPTM te converteren, moet u een presentatie maken van het bestand PPS en dit opslaan als PPTM.

{{% blocks/products/pf/agp/code-block title="Node.js-code voor het converteren van PPS naar PPTM" offSpacer="true" %}}

```javascript

var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var pres = new aspose.slides.Presentation("welcome-to-powerpoint.pps");
try
{
    pres.save("output.pptm", aspose.slides.SaveFormat.Pptm);
}
finally
{
    if (pres != null) pres.dispose();
}
```
