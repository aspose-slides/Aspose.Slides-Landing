Om het bestand PPSX naar PPSM te converteren, moet u een presentatie maken van het bestand PPSX en dit opslaan als PPSM.

{{% blocks/products/pf/agp/code-block title="Node.js-code voor het converteren van PPSX naar PPSM" offSpacer="true" %}}

```javascript

var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var pres = new aspose.slides.Presentation("welcome-to-powerpoint.ppsx");
try
{
    pres.save("output.ppsm", aspose.slides.SaveFormat.Ppsm);
}
finally
{
    if (pres != null) pres.dispose();
}
```
