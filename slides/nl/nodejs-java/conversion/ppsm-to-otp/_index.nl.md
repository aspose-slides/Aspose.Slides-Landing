Om het bestand PPSM naar OTP te converteren, moet u een presentatie maken van het bestand PPSM en dit opslaan als OTP.

{{% blocks/products/pf/agp/code-block title="Node.js-code voor het converteren van PPSM naar OTP" offSpacer="true" %}}

```javascript

var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var pres = new aspose.slides.Presentation("welcome-to-powerpoint.ppsm");
try
{
    pres.save("output.otp", aspose.slides.SaveFormat.Otp);
}
finally
{
    if (pres != null) pres.dispose();
}
```
