Om het bestand ODP naar PPT te converteren, moet u een presentatie maken van het bestand ODP en dit opslaan als PPT.

{{% blocks/products/pf/agp/code-block title="Node.js-code voor het converteren van ODP naar PPT" offSpacer="true" %}}

```javascript

var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var pres = new aspose.slides.Presentation("welcome-to-powerpoint.odp");
try
{
    pres.save("output.ppt", aspose.slides.SaveFormat.Ppt);
}
finally
{
    if (pres != null) pres.dispose();
}
```
