Om het bestand POT naar GIF te converteren, moet u een presentatie maken van het bestand POT en dit opslaan als GIF.

{{% blocks/products/pf/agp/code-block title="Node.js-code voor het converteren van POT naar GIF" offSpacer="true" %}}

```javascript

var aspose = aspose || {};

aspose.slides = require("aspose.slides.via.java");

var pres = new aspose.slides.Presentation("welcome-to-powerpoint.pot");
try
{
    pres.save("output.gif", aspose.slides.SaveFormat.Gif);
}
finally
{
    if (pres != null) pres.dispose();
}
```
