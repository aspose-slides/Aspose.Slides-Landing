<p>
 Aspose.Slides voor Android via Java is een PowerPoint API voor Android-applicaties. Ontwikkelaars kunnen een breed scala aan presentatieverwerkingstaken uitvoeren, waaronder lezen, schrijven, wijzigen, samenvoegen, splitsen, converteren en manipuleren van presentaties zonder enige afhankelijkheid van Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides voor Android via Java ondersteunt bijna alle native PowerPoint-bestandsindelingen en OpenOffice-presentatie-indeling (ODP). Maak uw PowerPoint-viewer-app voor Android, evenals converter, editor of iets anders. Presentatie-elementen zoals SmartArt-vormen, MSO-diagrammen, ActiveX-componenten en OLE-frameobjecten zijn toegankelijk via de PowerPoint API voor beheer en manipulatie.
</p>
