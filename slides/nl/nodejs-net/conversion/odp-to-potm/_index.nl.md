Om het bestand ODP naar POTM te converteren, moet u een presentatie maken van het bestand ODP en dit opslaan als POTM.

{{% blocks/products/pf/agp/code-block title="JavaScript-code voor het converteren van ODP naar POTM" offSpacer="true" %}}

```javascript

const fs = require('fs');
const asposeSlides = require('aspose.slides.via.net');
const { Presentation, SaveFormat } = asposeSlides;
var pres = new Presentation("welcome-to-powerpoint.odp");
try
{
    pres.save("output.potm", SaveFormat.Potm);
}
finally
{
    if (pres != null) pres.dispose();
}
```
