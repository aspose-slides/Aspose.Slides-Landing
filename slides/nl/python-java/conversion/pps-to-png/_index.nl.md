Om het bestand PPS naar PNG te converteren, moet u een presentatie maken van het bestand PPS en dit opslaan als PNG.

{{% blocks/products/pf/agp/code-block title="Python-tutorial voor het converteren van PPS naar PNG" offSpacer="true" %}}

```python

import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat
from javax.imageio import ImageIO
from java.io import File

pres = Presentation("PowerPoint.pps");

for i in range(pres.getSlides().size()):
    buffImage = pres.getSlides().get_Item(i).getThumbnail(2, 2)
    ImageIO.write(buffImage, "PNG", File("slide" + str(i) + ".png"))

jpype.shutdownJVM()
```
