<p>
 Aspose.Slides برای اندروید از طریق جاوا یک پاورپوینت API برای برنامه های اندروید است. توسعه دهندگان می توانند طیف گسترده ای از وظایف پردازش ارائه از جمله خواندن، نوشتن، اصلاح، ادغام، تقسیم، تبدیل و دستکاری ارائه ها را بدون وابستگی به Microsoft PowerPoint انجام دهند.
</p>

<p>
 Aspose.Slides برای اندروید از طریق جاوا تقریباً از تمام فرمت های فایل پاورپوینت بومی و همچنین فرمت ارائه OpenOffice (ODP) پشتیبانی می کند. برنامه نمایش پاورپوینت خود را برای اندروید و همچنین مبدل، ویرایشگر یا هر برنامه دیگری ایجاد کنید. عناصر ارائه مانند اشکال SmartArt، نمودارهای MSO، اجزای ActiveX و اشیاء قاب OLE را می توان از طریق PowerPoint API برای مدیریت و دستکاری در دسترس قرار داد.
</p>
