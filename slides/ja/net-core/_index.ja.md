<p>
 Aspose.Slides for .NET Coreは、Windows、Linux、macOSなどのオペレーティングシステムでプレゼンテーション処理ソリューションを作成するために使用されます。プレゼンテーションとその要素（テキスト、表、グラフ、メディア、アニメーション、SmartArts、ActiveX要素）を作成および変更します。
</p>

<p>
 プレゼンテーションビューア、クリエーター、コンバーター、エディター、テンプレートコンストラクター、マネージャー、マージ、比較、スプリッター、プリンター、プロテクターなどでアプリケーションを拡張します。 C＃アプリケーションがプレゼンテーションコンテンツ（テキスト、表、グラフ、メディア、SmartArts、ActiveX）を変更できるようにします。
</p>

<p>
 Aspose.Slidesによる.NETCorePowerPoint APIは、コードの重複を回避し、.NETCoreプラットフォームのすべての機能を使用するユニークなチャンスです。 APIはPowerPointまたはOpenOfficeから独立して機能し、それらを必要としません。
</p>

<p>
</p>

    <h2 class="h2title">
     .NETCoreでプレゼンテーションを変換する
    </h2>
    <p>
     Aspose.Slides for .NET Core APIは、PowerPointプレゼンテーション形式をドキュメント、画像、Web、またはファイル形式に変換できます。一般的な変換形式には、PDF、HTML、JPEG、XPSなどがあります。プレゼンテーション形式は、相互に変換することも（PPTからPPTX、PPTXからODPなど）、テンプレート形式（PPTXからPOTMなど）、スライドショー形式（PPTXからPPSXなど）に変換することも、その逆も可能です。変換されたファイルの品質は、PowerPoint .NETCoreAPI用のAspose.Slidesによって開発されたレンダリングエンジンのおかげで最高です。 Windows、Linux、macOSなど、どのOSでもプレゼンテーションを変換できます。
    </p>
    <div class="codeblock" id="code">
     <h3>
      プレゼンテーションのレンダリング-C＃
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>

    </p>

    </p>
