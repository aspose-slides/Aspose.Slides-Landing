        <h2 class="h2title">Jak vytvořit novou prezentaci v PowerPointu v Pythonu</h2>
        <p>V níže uvedeném příkladu jsme na první snímek prezentace přidali obdélník.</p>
        <pre>
            <code class="python">	
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat, ShapeType

pres = Presentation()
slide = pres.getSlides().get_Item(0)
slide.getShapes().addAutoShape(ShapeType.Rectangle, 50, 150, 300, 200)
pres.save("NewPresentation.pptx", SaveFormat.Pptx)

jpype.shutdownJVM()
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Jak sloučit prezentace v Pythonu</h2>
        <p>Tento kód Pythonu ukazuje, jak sloučit prezentace:</p>
        <pre>
            <code class="python">
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat

pres1 = Presentation("pres1.pptx");
pres2 = Presentation("pres2.pptx");

for i in range(pres2.getSlides().size()):
    pres1.getSlides().addClone(pres2.getSlides().get_Item(i));

pres1.save("combinedPresentation.pptx", SaveFormat.Pptx);

jpype.shutdownJVM()
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Jak importovat prezentaci z PDF v Pythonu. Převést PDF na PPT, Převést PDF na PPTX, Převést PDF na ODP</h2>
        <p>Tento kód Pythonu demonstruje proces převodu PDF do PowerPointu:</p>
        <pre>
            <code class="python">
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat

pres = Presentation();

pres.getSlides().removeAt(0);
pres.getSlides().addFromPdf("welcome-to-powerpoint.pdf");

pres.save("outputPresentation.pptx", SaveFormat.Pptx);

jpype.shutdownJVM()
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Jak převést PowerPoint do PDF v Pythonu</h2>
        <p>Tento kód Pythonu ukazuje, jak převést dokument PowerPoint PPT, PPTX a OpenOffice ODP na dokument PDF pomocí výchozích možností. Výsledný soubor je dokument PDF v maximální kvalitě</p>
        <pre>
            <code class="python">
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat

pres = Presentation("PowerPoint.pptx");

pres.save("PPTX-to-PDF.pdf", SaveFormat.Pdf);

jpype.shutdownJVM()
            </code>
        </pre>
    </div>
    <div class="col-lg-12">
        <h2 class="h2title">Jak převést PowerPoint do JPG v Pythonu. Převést PPT na JPG, Převést PPTX na JPG, Převést ODP na JPG</h2>
        <p>Následující příklad ukazuje, jak převést PowerPoint PPT, PPTX a dokument ODP OpenOffice na sadu obrázků JPEG.</p>
        <pre>
            <code class="python">
import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation
from javax.imageio import ImageIO
from java.io import File

pres = Presentation("pres.pptx");

format_name = "PNG"

for i in range(pres.getSlides().size()):
    buffImage = pres.getSlides().get_Item(i).getThumbnail(2, 2)
    ImageIO.write(buffImage, format_name, File("image_java" + str(i) + ".png"))

jpype.shutdownJVM()
            </code>
        </pre>
