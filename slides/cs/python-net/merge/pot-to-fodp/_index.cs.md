{{% blocks/products/pf/feature-page-section  h2="Sloučit soubory POT do FODP pomocí Pythonu" %}}
Chcete-li sloučit prezentace PowerPoint, budete muset naklonovat snímky z jedné prezentace do druhé.

{{% blocks/products/pf/agp/code-block title="Python kód pro sloučení více POT do jednoho souboru FODP" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.pot") as pres1:
    with slides.Presentation("presentation2.pot") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.fodp", slides.export.SaveFormat.FODP)
```
