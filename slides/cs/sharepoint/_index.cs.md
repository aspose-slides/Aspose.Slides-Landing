<p>
 Aspose.Slides pro SharePoint pro správu a převod souborů PowerPoint v SharePointu. Aspose.Slides, jako řešení SharePoint PowerPoint, může snadno importovat běžné prezentační formáty společnosti Microsoft včetně PPT, POT, PPS, PPTX, POTX & PPSX a může tyto formáty exportovat do souborů PDF, TIFF a XPS. Používejte všechny funkce SharePoint PPT bez nutnosti automatizace aplikace Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides for SharePoint funguje se službou Windows SharePoint Services WSS 3.0 a Microsoft Office SharePoint Server 2007, 2010 a 2013.
</p>
