Per unire le presentazioni PowerPoint, dovrai clonare le diapositive da una presentazione all'altra.

{{% blocks/products/pf/agp/code-block title="Codice Python per unire più PPSX in un singolo file TIFF" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.ppsx") as pres1:
    with slides.Presentation("presentation2.ppsx") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.tiff", slides.export.SaveFormat.TIFF)
```
