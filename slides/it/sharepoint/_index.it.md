<p>
 Aspose.Slides per SharePoint per gestire e convertire file PowerPoint in SharePoint. Aspose.Slides, come soluzione di SharePoint PowerPoint, può importare facilmente formati di presentazione Microsoft comuni tra cui PPT, POT, PPS, PPTX, POTX e PPSX ed esportare questi formati in file PDF, TIFF e XPS. Utilizza tutte le funzionalità PPT di SharePoint senza l'automazione di Microsoft PowerPoint necessaria.
</p>

<p>
 Aspose.Slides per SharePoint funziona con Windows SharePoint Services WSS 3.0 e Microsoft Office SharePoint Server 2007, 2010 e 2013.
</p>
