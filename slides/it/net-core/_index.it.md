<p>
 Aspose.Slides per .NET Core viene utilizzato per creare soluzioni di elaborazione di presentazioni su qualsiasi sistema operativo: Windows, Linux o macOS. Crea e modifica la presentazione e i suoi elementi: testo, tabelle, grafici, media, animazioni, SmartArts ed elementi ActiveX.
</p>

<p>
 Estendi le tue applicazioni con visualizzatore di presentazioni, creatore, convertitore, editor, costruttore di modelli, manager, fusione, comparatore, splitter, stampante, protezione e altri. Consenti alle applicazioni C# di modificare qualsiasi contenuto di presentazione: testi, tabelle, grafici, media, SmartArts, ActiveX.
</p>

<p>
 L'API .NET Core PowerPoint di Aspose.Slides è un'opportunità unica per evitare la duplicazione del codice e utilizzare tutta la potenza della piattaforma .NET Core. L'API funziona indipendentemente da PowerPoint o Open Office e non li richiede.
</p>

</p>

    <h2 class="h2title">
     Converti presentazione in .NET Core
    </h2>
    <p>
     Aspose.Slides per .NET Core API può convertire qualsiasi formato di presentazione PowerPoint in un documento, immagine, Web o formato di file. Alcuni dei formati di conversione più diffusi sono: PDF, HTML, JPEG, XPS. I formati di presentazione possono anche essere convertiti tra loro (ad es. da PPT a PPTX, da PPTX a ODP) o in un formato modello (ad es. da PPTX a POTM), un formato di presentazione (ad es. da PPTX a PPSX) e viceversa. La qualità del file convertito è la più alta grazie a un motore di rendering sviluppato da Aspose.Slides per PowerPoint .NET Core API. Puoi essere sicuro di convertire qualsiasi presentazione su qualsiasi sistema operativo: Windows, Linux o macOS.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Presentazioni di rendering - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Modifica presentazione in .NET Core
    </h2>
    <p>
     Con Aspose.Slides per .NET Core puoi modificare la logica e l'aspetto della presentazione. Tutto il contenuto testuale può essere modificato tramite elementi come testo, paragrafo, porzione, tabella, nota, commento, simbolo matematico. Sono disponibili effetti e animazioni 2D, 3D per forme, grafici, SmartArt, WordArt. Il design delle presentazioni può essere creato utilizzando modello, tema, schema diapositiva, layout diapositiva. Media, filigrana, ActiveX sono accessibili anche tramite l'API PowerPoint .NET Core di Aspose.Slides.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Unisci, confronta, dividi, stampa presentazione in .NET Core
    </h2>
    <p>
     Aspose.Slides per .NET Core supporta molte funzionalità aggiuntive non fornite da PowerPoint e altri software di terze parti per le presentazioni. Tra questi ci sono il confronto di presentazioni e modelli, la suddivisione della presentazione in base alla logica personalizzata, l'unione di presentazioni di vari formati, l'unione/applicazione di modelli in una serie di presentazioni, la stampa di presentazioni, ecc.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Microsoft Office Automation: non necessario

È una perfetta alternativa all'automazione di Microsoft Office in termini di funzionalità supportate, sicurezza, stabilità, scalabilità, velocità e prezzo.
