<p>
 Aspose.Slides per Android tramite Java è un'API di PowerPoint per applicazioni Android. Gli sviluppatori possono eseguire un'ampia gamma di attività di elaborazione delle presentazioni, tra cui leggere, scrivere, modificare, unire, dividere, convertire e manipolare le presentazioni senza alcuna dipendenza da Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides per Android tramite Java supporta quasi tutti i formati di file PowerPoint nativi e il formato di presentazione OpenOffice (ODP). Crea la tua app di visualizzazione di PowerPoint per Android, nonché convertitore, editor o qualsiasi altro. È possibile accedere a elementi di presentazione come forme SmartArt, grafici MSO, componenti ActiveX e oggetti frame OLE tramite l'API di PowerPoint per la gestione e la manipolazione.
</p>
