<p>
 Aspose.Slides for Android عبر Java هو PowerPoint API لتطبيقات Android. يمكن للمطورين أداء مجموعة واسعة من مهام معالجة العروض التقديمية بما في ذلك القراءة والكتابة والتعديل والدمج والتقسيم وتحويل ومعالجة العروض التقديمية دون أي اعتماد على Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides for Android عبر Java يدعم تقريبًا جميع تنسيقات ملفات PowerPoint الأصلية بالإضافة إلى تنسيق عرض OpenOffice (ODP). قم بإنشاء تطبيق عارض PowerPoint الخاص بك لنظام Android ، بالإضافة إلى المحول أو المحرر أو أي تطبيق آخر. يمكن الوصول إلى عناصر العرض التقديمي مثل أشكال SmartArt ومخططات MSO ومكونات ActiveX وكائنات إطار OLE عبر PowerPoint API للإدارة والمعالجة.
</p>
