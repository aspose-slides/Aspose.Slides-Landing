<p>
 Aspose.Slides pour SharePoint pour gérer et convertir des fichiers PowerPoint dans SharePoint. Aspose.Slides, en tant que solution SharePoint PowerPoint, peut facilement importer des formats de présentation Microsoft courants, notamment PPT, POT, PPS, PPTX, POTX et PPSX, et peut exporter ces formats vers des fichiers PDF, TIFF et XPS. Utilisez toutes les fonctionnalités SharePoint PPT sans Microsoft PowerPoint Automation nécessaire.
</p>

<p>
 Aspose.Slides pour SharePoint fonctionne avec Windows SharePoint Services WSS 3.0 et Microsoft Office SharePoint Server 2007, 2010 et 2013.
</p>
