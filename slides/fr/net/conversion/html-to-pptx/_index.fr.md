En utilisant [**Aspose.Slides pour .NET**](https://products.aspose.com/slides/fr/net/), vous pouvez convertir un document HTML en présentation PowerPoint avec seulement quelques lignes de code :

{{% blocks/products/pf/agp/code-block title="Code C# pour convertir HTML en PPTX" offSpacer="true" %}}
```cs
using (var presentation = new Presentation())
{
    using (var htmlStream = File.OpenRead("page.html"))
    {
        presentation.Slides.AddFromHtml(htmlStream);
    }

    presentation.Save("Presentation.pptx", SaveFormat.Pptx);
}
```
