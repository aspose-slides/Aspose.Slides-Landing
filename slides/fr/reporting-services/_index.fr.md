<p>
 Aspose.Slides for Reporting Services est la seule solution sur le marché qui permet de générer des présentations dans les formats natifs de PowerPoint à partir de Microsoft SQL Server Reporting Services.
</p>

<p>
 Aspose.Slides for Reporting Services prend en charge presque toutes les fonctionnalités RDL telles que les tableaux, les matrices, les graphiques, les images, les listes, les sous-rapports et les en-têtes / pieds de page, permettant ainsi d'exporter des rapports dans des formats de présentation PowerPoint de manière à ce qu'ils ressemblent à l'original. conception du rapport. Exécutez Reporting Services SQL PPT sans Microsoft PowerPoint Automation.
</p>
