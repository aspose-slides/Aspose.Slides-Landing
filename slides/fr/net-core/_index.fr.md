<p>
 Aspose.Slides pour .NET Core est utilisé pour créer des solutions de traitement de présentation sur n'importe quel système d'exploitation : Windows, Linux ou macOS. Créez et modifiez la présentation et ses éléments : texte, tableaux, graphiques, médias, animations, éléments SmartArts et ActiveX.
</p>

<p>
 Étendez vos applications avec un visualiseur de présentation, un créateur, un convertisseur, un éditeur, un constructeur de modèles, un gestionnaire, une fusion, un comparateur, un séparateur, une imprimante, un protecteur et autres. Autorisez les applications C# à modifier n'importe quel contenu de présentation : textes, tableaux, graphiques, médias, SmartArts, ActiveX.
</p>

<p>
 L'API .NET Core PowerPoint par Aspose.Slides est une chance unique d'éviter la duplication de code et d'utiliser toute la puissance de la plate-forme .NET Core. L'API fonctionne indépendamment de PowerPoint ou d'Open Office et n'en a pas besoin.
</p>

   <div class="col-lg-12">
    <h2 class="h2title">
     Convertir la présentation dans .NET Core
    </h2>
    <p>
     L'API Aspose.Slides pour .NET Core peut convertir n'importe quel format de présentation PowerPoint en format de document, d'image, Web ou de fichier. Certains des formats de conversion populaires sont : PDF, HTML, JPEG, XPS. Les formats de présentation peuvent également être convertis entre eux (par exemple, PPT en PPTX, PPTX en ODP), ou en un format de modèle (par exemple, PPTX en POTM), un format de diaporama (par exemple, PPTX en PPSX) et vice versa. La qualité du fichier converti est la plus élevée grâce à un moteur de rendu développé par Aspose.Slides pour PowerPoint .NET Core API. Vous pouvez être sûr de convertir n'importe quelle présentation sur n'importe quel système d'exploitation : Windows, Linux ou macOS.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Présentations de rendu - C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Modifier la présentation dans .NET Core
    </h2>
    <p>
     Avec Aspose.Slides pour .NET Core, vous pouvez modifier la logique et l'apparence de la présentation. Tout le contenu textuel peut être modifié via des éléments tels que texte, paragraphe, portion, tableau, note, commentaire, symbole mathématique. Des effets et des animations 2D, 3D sont disponibles pour les formes, les graphiques, les SmartArts, les WordArts. La conception des présentations peut être créée à l'aide d'un modèle, d'un thème, d'un masque de diapositives, d'une mise en page de diapositives. Média, filigrane, ActiveX sont également accessibles via l'API PowerPoint .NET Core par Aspose.Slides.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Fusionner, comparer, diviser, imprimer la présentation dans .NET Core
    </h2>
    <p>
     Aspose.Slides pour .NET Core prend en charge de nombreuses fonctionnalités supplémentaires non fournies par PowerPoint et d'autres logiciels tiers pour les présentations. Parmi eux, on peut comparer des présentations et des modèles, diviser la présentation en fonction de la logique personnalisée, fusionner des présentations de différents formats, fusionner/appliquer un modèle dans un ensemble de présentations, imprimer des présentations, etc.
    </p>
   </div>

     Microsoft Office Automation - Non nécessaire

C'est une alternative parfaite à Microsoft Office Automation en termes de fonctionnalités prises en charge, de sécurité, de stabilité, d'évolutivité, de vitesse et de prix.
