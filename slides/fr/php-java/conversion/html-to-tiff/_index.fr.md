Pour convertir le HTML en TIFF, vous devrez créer une présentation à partir du fichier HTML et l'enregistrer sous le nom TIFF.

{{% blocks/products/pf/agp/code-block title="Code PHP pour convertir HTML en TIFF" offSpacer="true" %}}

```php

<?php
require_once("http://localhost:8080/JavaBridge/java/Java.inc");
require_once("lib/aspose.slides.php");
        
$pres = new Presentation();
try
{
    $pres->getSlides()->removeAt(0);
    
    $filename = 'file.html';
    $f = fopen($filename, 'r');
    if ($f) {
        $contents = fread($f, filesize($filename));
        fclose($f);
    }
    
    $pres->getSlides()->addFromHtml($contents);        
    $pres->save("output.tiff", SaveFormat::Tiff);        
}
finally
{
    if ($pres != null) $pres->dispose();
}
?>
```
