{{% blocks/products/pf/feature-page-section  h2="Fusionner les fichiers PPS à l'aide de Python" %}}
Pour fusionner les présentations PowerPoint, vous devrez cloner les diapositives d'une présentation à l'autre.

{{% blocks/products/pf/agp/code-block title="Code Python pour fusionner plusieurs PPS en un seul fichier" offSpacer="true" %}}


```python

import aspose.slides as slides


with slides.Presentation("presentation1.pps") as pres1:
    with slides.Presentation("presentation2.pps") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.pps", slides.export.SaveFormat.PPS)
```
