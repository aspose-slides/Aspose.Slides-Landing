<p>
 Aspose.Slides for SharePoint 用於管理和轉換 SharePoint 中的 PowerPoint 文件。 Aspose.Slides 作為 SharePoint PowerPoint 解決方案，可以輕鬆導入常見的 Microsoft 演示格式，包括 PPT、POT、PPS、PPTX、POTX 和 PPSX，並且可以將這些格式導出為 PDF、TIFF 和 XPS 文件。無需 Microsoft PowerPoint 自動化即可使用所有 SharePoint PPT 功能。
</p>

<p>
 Aspose.Slides for SharePoint 適用於 Windows SharePoint Services WSS 3.0 和 Microsoft Office SharePoint Server 2007、2010 和 2013。
</p>
