<p>
 Aspose.Slides for Reporting Services 是市場上唯一可以在 Microsoft SQL Server Reporting Services 中生成 PowerPoint 原生格式的演示文稿的解決方案。
</p>

<p>
 Aspose.Slides for Reporting Services 支持幾乎所有 RDL 功能，如表格、矩陣、圖表、圖像、列表、子報告和頁眉/頁腳，因此允許以 PowerPoint 演示文稿格式導出報告，使其看起來與原始格式相似報告設計。在沒有 Microsoft PowerPoint 自動化的情況下執行 Reporting Services SQL PPT。
</p>
