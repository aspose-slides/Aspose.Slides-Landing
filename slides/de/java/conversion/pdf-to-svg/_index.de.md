{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie PDF in SVG mit Java" %}}
Um das PDF in SVG zu konvertieren, müssen Sie eine Präsentation aus der PDF-Datei erstellen und sie als SVG speichern.

{{% blocks/products/pf/agp/code-block title="Java-Code zum Konvertieren von PDF in SVG" offSpacer="true" %}}

```java

Presentation pres = new Presentation();
try {
    pres.getSlides().addFromPdf("InputPDF.pdf");
    for (int index = 0; index < pres.getSlides().size(); index++)
    {
        ISlide slide = pres.getSlides().get_Item(index);

        FileOutputStream fileStream = new FileOutputStream("slide-" + index + ".svg");
        try {
            slide.writeAsSvg(fileStream);
        } finally {
            fileStream.close();
        }
    }
} finally {
    if (pres != null) pres.dispose();
}
```
