{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie PDF in PPTX mit Java" %}}
Um das PDF in PPTX zu konvertieren, müssen Sie eine Präsentation aus der PDF-Datei erstellen und sie als PPTX speichern.

{{% blocks/products/pf/agp/code-block title="Java-Code zum Konvertieren von PDF in PPTX" offSpacer="true" %}}

```java

Presentation pres = new Presentation();
try {
    pres.getSlides().addFromPdf("InputPDF.pdf");
    pres.save("OutputPresentation.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
