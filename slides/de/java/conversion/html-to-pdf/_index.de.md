{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie HTML in PDF mit Java" %}}
Um das HTML in PDF zu konvertieren, müssen Sie eine Präsentation aus der HTML-Datei erstellen und sie als PDF speichern.

{{% blocks/products/pf/agp/code-block title="Java-Code zum Konvertieren von HTML in PDF" offSpacer="true" %}}

```java

Presentation pres = new Presentation();
try {
    TextReader tr = new StreamReader("file.html");
    pres.getSlides().addFromHtml(tr);
    pres.save("index.pdf", SaveFormat.Pdf);
} finally {
    if (pres != null) pres.dispose();
}
```
