{{% blocks/products/pf/feature-page-section  h2="Konvertieren Sie JPG in PNG mit Java" %}}
Um das JPG in PNG zu konvertieren, müssen Sie eine Präsentation aus der JPG-Datei erstellen und sie als PNG speichern.

{{% blocks/products/pf/agp/code-block title="Java-Code zum Konvertieren von JPG in PNG" offSpacer="true" %}}

```java

Presentation pres = new Presentation();
try {
    ISlide slide = pres.getSlides().get_Item(0);
	IPPImage image = pres.getImages().addImage(Files.readAllBytes(Paths.get("image.jpg")));
	slide.getShapes().addPictureFrame(ShapeType.Rectangle, 10, 10, 100, 100, image);
    Dimension size = new Dimension(960, 720);
    for (int index = 0; index < pres.getSlides().size(); index++)
    {
        ISlide slide = pres.getSlides().get_Item(index);
        BufferedImage bufferedImage = slide.getThumbnail(size);
        ImageIO.write(bufferedImage, "PNG", new File("image_java_" + index + ".png"));
    }
} finally {
    if (pres != null) pres.dispose();
}
```
