Um das ODP in POTX zu konvertieren, müssen Sie eine Präsentation aus der ODP-Datei erstellen und diese als POTX speichern.

{{% blocks/products/pf/agp/code-block title="JavaScript-Code zum Konvertieren von ODP in POTX" offSpacer="true" %}}

```javascript

const fs = require('fs');
const asposeSlides = require('aspose.slides.via.net');
const { Presentation, SaveFormat } = asposeSlides;
var pres = new Presentation("welcome-to-powerpoint.odp");
try
{
    pres.save("output.potx", SaveFormat.Potx);
}
finally
{
    if (pres != null) pres.dispose();
}
```
