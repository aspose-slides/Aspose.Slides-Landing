Um das PPS in GIF zu konvertieren, müssen Sie eine Präsentation aus der PPS-Datei erstellen und diese als GIF speichern.

{{% blocks/products/pf/agp/code-block title="JavaScript-Code zum Konvertieren von PPS in GIF" offSpacer="true" %}}

```javascript

const fs = require('fs');
const asposeSlides = require('aspose.slides.via.net');
const { Presentation, SaveFormat } = asposeSlides;
var pres = new Presentation("welcome-to-powerpoint.pps");
try
{
    pres.save("output.gif", SaveFormat.Gif);
}
finally
{
    if (pres != null) pres.dispose();
}
```
