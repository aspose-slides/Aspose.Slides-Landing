Um das POT in PPS zu konvertieren, müssen Sie eine Präsentation aus der POT-Datei erstellen und diese als PPS speichern.

{{% blocks/products/pf/agp/code-block title="JavaScript-Code zum Konvertieren von POT in PPS" offSpacer="true" %}}

```javascript

const fs = require('fs');
const asposeSlides = require('aspose.slides.via.net');
const { Presentation, SaveFormat } = asposeSlides;
var pres = new Presentation("welcome-to-powerpoint.pot");
try
{
    pres.save("output.pps", SaveFormat.Pps);
}
finally
{
    if (pres != null) pres.dispose();
}
```
