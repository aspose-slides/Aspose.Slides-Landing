Um das PPT in OTP zu konvertieren, müssen Sie eine Präsentation aus der PPT-Datei erstellen und diese als OTP speichern.

{{% blocks/products/pf/agp/code-block title="JavaScript-Code zum Konvertieren von PPT in OTP" offSpacer="true" %}}

```javascript

const fs = require('fs');
const asposeSlides = require('aspose.slides.via.net');
const { Presentation, SaveFormat } = asposeSlides;
var pres = new Presentation("welcome-to-powerpoint.ppt");
try
{
    pres.save("output.otp", SaveFormat.Otp);
}
finally
{
    if (pres != null) pres.dispose();
}
```
