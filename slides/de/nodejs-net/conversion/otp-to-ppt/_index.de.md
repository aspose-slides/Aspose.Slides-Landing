Um das OTP in PPT zu konvertieren, müssen Sie eine Präsentation aus der OTP-Datei erstellen und diese als PPT speichern.

{{% blocks/products/pf/agp/code-block title="JavaScript-Code zum Konvertieren von OTP in PPT" offSpacer="true" %}}

```javascript

const fs = require('fs');
const asposeSlides = require('aspose.slides.via.net');
const { Presentation, SaveFormat } = asposeSlides;
var pres = new Presentation("welcome-to-powerpoint.otp");
try
{
    pres.save("output.ppt", SaveFormat.Ppt);
}
finally
{
    if (pres != null) pres.dispose();
}
```
