Um das POTM in JPG zu konvertieren, müssen Sie eine Präsentation aus der POTM-Datei erstellen und diese als JPG speichern.

{{% blocks/products/pf/agp/code-block title="Python-Tutorial zum Konvertieren von POTM in JPG" offSpacer="true" %}}

```python

import jpype
import asposeslides

jpype.startJVM()

from asposeslides.api import Presentation, SaveFormat
from javax.imageio import ImageIO
from java.io import File

pres = Presentation("PowerPoint.potm");

for i in range(pres.getSlides().size()):
    buffImage = pres.getSlides().get_Item(i).getThumbnail(2, 2)
    ImageIO.write(buffImage, "JPEG", File("slide" + str(i) + ".jpg"))

jpype.shutdownJVM()
```
