<p>
 Aspose.Slides for Xamarin ist eine plattformübergreifende Lösung zum Erstellen von mobilen PowerPoint-Anwendungen für jedes Gerät und Betriebssystem. Erlauben Sie Ihrer mobilen Anwendung, Text, Medien und Formen der Folie zu bearbeiten, und speichern Sie die Präsentation im gewünschten Format. Überprüfen Sie die Präsentation auf dem Handy, fügen Sie Kommentare und Notizen hinzu und senden Sie sie. Wenden Sie ein neues Thema auf die Präsentation an, teilen Sie die Präsentation auf mehrere Dateien auf.
</p>

<p>
 Aspose.Slides for Xamarin ist eine eigenständige Xamarin PowerPoint-API, die weder Microsoft PowerPoint noch andere zusätzliche Software benötigt.
</p>

    <h2 class="h2title">
     Konvertieren Sie die Präsentation über die Xamarin PowerPoint-API
    </h2>
    <p>
     Aspose.Slides for Xamarin API ermöglicht das Erstellen mobiler Anwendungen für die PowerPoint-Konvertierung in beliebige Dokument-, Text-, Bild-, Web- oder andere Formate. Konvertieren Sie Präsentationen in hoher Qualität sogar von einem Handy aus. Konvertieren Sie Präsentationen in das PDF-, HTML-, XPS- und JPG-Format. Die PowerPoint Xamarin-API ist einfach zu verwenden, die Konvertierungslogik ist in einem 1-2-zeiligen Code implementiert. Laden Sie einfach eine Präsentation und speichern Sie sie im gewünschten Format.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Präsentationen rendern – C#
     </h3>
     <pre><code class="cs">// load the file to be converted

var prs = new Aspose.Slides.Presentation(dir + "template.pptx");

// render as PDF, XPS &amp; TIFF

prs.Save(dir + "output.pdf", Aspose.Slides.Export.SaveFormat.Pdf);

prs.Save(dir + "output.xps", Aspose.Slides.Export.SaveFormat.Xps);

prs.Save(dir + "output.tiff", Aspose.Slides.Export.SaveFormat.Tiff);</code></pre>
    </div>
   </div>

     Ändern Sie die Präsentation über die Xamarin PowerPoint-API
    </h2>
    <p>
     Mit der Xamarin PowerPoint-API von Aspose ist es möglich, beliebige Folien- und Präsentationsinhalte sowie Metadaten zu ändern. Auf Text, Tabellen, Formen, Diagramme, SmartArts und Animationen kann über die mobile Xamarin-Anwendung zugegriffen werden. Alle für Präsentationsinhalte verfügbaren Vorgänge können über die Aspose.Slides Xamarin-API angewendet werden. Präsentationskommentare, Notizen, Dokumenteigenschaften sowie Layouts, Themen und Vorlagen können ebenfalls geändert werden.
    </p>

    </h2>
    <p>
     Erstellen Sie ein Präsentationsverarbeitungssystem in Xamarin mit der Aspose.Slides Xamarin-API. Solche Operationen über Präsentationen und Folien jeglicher Art sind verfügbar: Zusammenführen, Teilen, Vergleichen, Drucken usw. Die Operationen können verwendet werden, um viele Verarbeitungsszenarien zu erstellen. Beispielsweise kann die Zusammenführungsfunktion verwendet werden, um eine Vorlage auf eine Präsentation anzuwenden, einige Vorlagen zusammenzuführen, Präsentationen verschiedener Formate in einem Format zusammenzuführen usw.
    </p>
