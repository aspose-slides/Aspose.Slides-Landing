{{% blocks/products/pf/feature-page-section  h2="使用 Python 将 PPTM 文件合并到 HTML" %}}
要合并 PowerPoint 演示文稿，您需要将幻灯片从一个演示文稿克隆到另一个演示文稿。

{{% blocks/products/pf/agp/code-block title="用于将多个 PPTM 合并为单个 HTML 文件的 Python 代码" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.pptm") as pres1:
    with slides.Presentation("presentation2.pptm") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.html", slides.export.SaveFormat.HTML5)
```
