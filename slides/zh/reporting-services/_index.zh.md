<p>
 Aspose.Slides for Reporting Services 是市场上唯一可以在 Microsoft SQL Server Reporting Services 中生成 PowerPoint 原生格式的演示文稿的解决方案。
</p>

<p>
 Aspose.Slides for Reporting Services 支持几乎所有 RDL 功能，如表格、矩阵、图表、图像、列表、子报告和页眉/页脚，因此允许以 PowerPoint 演示文稿格式导出报告，使其看起来与原始格式相似报告设计。在没有 Microsoft PowerPoint 自动化的情况下执行 Reporting Services SQL PPT。
</p>
