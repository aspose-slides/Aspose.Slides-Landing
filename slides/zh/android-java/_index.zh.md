<p>
 Aspose.Slides for Android via Java 是适用于 Android 应用程序的 PowerPoint API。开发人员可以执行广泛的演示文稿处理任务，包括阅读、编写、修改、合并、拆分、转换和操作演示文稿，而无需依赖 Microsoft PowerPoint。
</p>

<p>
 Aspose.Slides for Android via Java 支持几乎所有原生 PowerPoint 文件格式以及 OpenOffice 演示文稿格式 (ODP)。创建适用于 Android 的 PowerPoint 查看器应用程序，以及转换器、编辑器或任何其他应用程序。 SmartArt 形状、MSO 图表、ActiveX 组件和 OLE 框架对象等演示元素可以通过 PowerPoint API 进行访问以进行管理和操作。
</p>
