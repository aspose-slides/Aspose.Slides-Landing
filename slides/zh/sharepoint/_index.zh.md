<p>
 Aspose.Slides for SharePoint 用于管理和转换 SharePoint 中的 PowerPoint 文件。 Aspose.Slides 作为 SharePoint PowerPoint 解决方案，可以轻松导入常见的 Microsoft 演示格式，包括 PPT、POT、PPS、PPTX、POTX 和 PPSX，并且可以将这些格式导出为 PDF、TIFF 和 XPS 文件。无需 Microsoft PowerPoint 自动化即可使用所有 SharePoint PPT 功能。
</p>

<p>
 Aspose.Slides for SharePoint 适用于 Windows SharePoint Services WSS 3.0 和 Microsoft Office SharePoint Server 2007、2010 和 2013。
</p>
