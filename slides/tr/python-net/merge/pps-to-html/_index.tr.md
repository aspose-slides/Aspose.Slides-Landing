{{% blocks/products/pf/agp/code-block title="Birden çok PPS dosyasını tek bir HTML dosyasına birleştirmek için Python kodu" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.pps") as pres1:
    with slides.Presentation("presentation2.pps") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.html", slides.export.SaveFormat.HTML5)
```
