{{% blocks/products/pf/agp/code-block title="Birden çok POTM dosyasını tek bir TIFF dosyasına birleştirmek için Python kodu" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.potm") as pres1:
    with slides.Presentation("presentation2.potm") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.tiff", slides.export.SaveFormat.TIFF)
```
