Aby scalić prezentacje PowerPoint, musisz sklonować slajdy z jednej prezentacji do drugiej.

{{% blocks/products/pf/agp/code-block title="Kod Pythona do łączenia wielu plików ODP w jeden plik TIFF" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.odp") as pres1:
    with slides.Presentation("presentation2.odp") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.tiff", slides.export.SaveFormat.TIFF)
```
