Aby scalić prezentacje PowerPoint, musisz sklonować slajdy z jednej prezentacji do drugiej.

{{% blocks/products/pf/agp/code-block title="Kod Pythona do łączenia wielu plików POTM w jeden plik POTX" offSpacer="true" %}}

```python

import aspose.slides as slides


with slides.Presentation("presentation1.potm") as pres1:
    with slides.Presentation("presentation2.potm") as pres2:
        for slide in pres2.slides:
            pres1.slides.add_clone(slide)
    pres1.save("presentation.potx", slides.export.SaveFormat.POTX)
```
