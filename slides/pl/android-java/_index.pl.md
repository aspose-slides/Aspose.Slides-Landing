<p>
 Aspose.Slides dla systemu Android przez Javę to interfejs API programu PowerPoint dla aplikacji na Androida. Programiści mogą wykonywać szeroki zakres zadań związanych z przetwarzaniem prezentacji, w tym czytać, pisać, modyfikować, scalać, dzielić, konwertować i manipulować prezentacjami bez jakiejkolwiek zależności od programu Microsoft PowerPoint.
</p>

<p>
 Aspose.Slides dla Androida za pośrednictwem Javy obsługuje prawie wszystkie natywne formaty plików PowerPoint, a także format prezentacji OpenOffice (ODP). Utwórz aplikację przeglądarki PowerPoint dla systemu Android, a także konwerter, edytor lub dowolną inną. Dostęp do elementów prezentacji, takich jak kształty SmartArt, wykresy MSO, składniki ActiveX i obiekty ramek OLE, można uzyskać za pośrednictwem interfejsu API programu PowerPoint w celu zarządzania i manipulacji.
</p>
